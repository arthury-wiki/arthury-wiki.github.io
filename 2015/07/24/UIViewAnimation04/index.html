UIBezierPath介绍

  • 此类是Core Graphics框架关于path的一个封装。使用此类可以定义简单的形状，如椭圆或者矩形，或者有多个直线和曲线段组成的形状；至于其他更加自定义的方法，可以考虑使用CGContextRef相关的更加底层的api来进行绘制，关于这一块可以去看一下Quartz2D框架
  • UIBezierPath对象是CGPathRef数据类型的封装。path可以由直线和曲线段去创建。我们使用直线段去创建矩形和多边形，使用曲线段去创建弧，圆或者其他更复杂的曲线形状
  • 每一个线段都包含有一个或者多个点，绘图命令定义如何去诠释这些点；每一个直线段或者曲线段结束的敌法就是下一个线段开始的地方
  • 每一个连接的直线或者曲线段的集合成为subpath；一个UIBezierPath对象定义一个完整的路径包括一个或者多个

基本的创建步骤

  • 首先要明确创建和使用一个path对象的过程是分开的
  • 创建一个path对象
    • 首先创建一个Bezier path对象
    • 使用方法moveToPoint:去设置初始线段的起点
    • 添加line或者是curve去定义一个或者对个subpaths
    • 改变UIBezierPath对象跟绘图相关的属性
  • 创建path的时候，我们英爱管理path上面的点相对于原点(0,0)，这样之后就可以很容易的移动path
  • 要绘制出path对象，就需要用到stroke和fill方法来在current graphic context下渲染path的line以及curve段

绘制一个多边形

  • 多边形由一些直线线条组成，可是使用moveToPoint:设置想要创建的形状的起点，从这点开始，可以使用方法addLineToPoint:
  • 可以连续创建line，每一个line的起点都是先前的终点，终点就是指定的点
  • closePath方法不仅可以结束一个shape的subpath描述，它同时在最后一个点和第一个点之间画一条线段，如果画多边形的话，这是一个便利的方法，可以不用专门去画最后一条线

UIBezierPath贝塞尔曲线的常用绘图方法

  • 矩形 + (UIBezierPath *)bezierPathWithRect:(CGRect)rect
  • 矩形内切椭圆 + (UIBezierPath *)bezierPathWithOvalInRect:(CGRect)rect
    • 其中参数中rect的width表示横长轴，height表示短长轴
  • 圆角矩形 + (UIBezierPath *)bezierPathWithRoundedRect:(CGRect)rect cornerRadius:(CGFloat)cornerRadius
    • 这是创建一个可以控制四个角圆滑度的矩形，conerRadius指的是四个角的圆半径，超过最大值之后就不会变化了
  • 可设置的圆角矩形 + (UIBezierPath *)bezierPathWithRoundedRect:(CGRect)rect byRoundingCorners:(UIRectCorner)corners cornerRadii:(CGSize)cornerRadii
    • 在矩形中，可以针对四角中的某个角加圆角，上下左右，或者是全部，cornerRadii是控制圆角的大小

  • 画圆 + (UIBezierPath *)bezierPathWithArcCenter:(CGPoint)center radius:(CGFloat)radius startAngle:(CGFloat)startAngle endAngle:(CGFloat)endAngle clockwise:(BOOL)clockwise

    • //圆的中心点，圆的半径，开始与结束的弧度，这个可以参考来确定圆上控制点的坐标，最后一个是顺逆时针

    • 关于计算弧度开始与技术的位置，这里有一个小技巧转换成角度来进行控制

      #define pi 3.14159265359
#define   DEGREES_TO_RADIANS(degrees)  ((pi * degrees)/ 180)
//这样就可以直接使用DEGREES_TO_RADIANS来输入角度了，非常的方便，已经放到snippet当中了

  • 画出相反的曲线 - bezierPathByReversingPath

    • 创建相反路径贝塞尔曲线对象，顺时针变逆时针，逆时针编程顺时针，对外观没有影响，但是对填充颜色的规则有影响
  • 自定义绘制线条组合
    • 起点：- (void)moveToPoint:(CGPoint)point
    • 直线：- (void)addLineToPoint:(CGPoint)point
    • 曲线：- (void)addCurveToPoint:(CGPoint)endPoint controlPoint1:(CGPoint)controlPoint1 controlPoint2:(CGPoint)controlPoint2，曲线回朝控制点方向起伏
    • 二次曲线：- (void)addQuadCurveToPoint:(CGPoint)endPoint controlPoint:(CGPoint)controlPoint
    • 三次曲线：- (void)addCurveToPoint:(CGPoint)endPoint controlPoint1:(CGPoint)controlPoint1 controlPoint2:(CGPoint)controlPoin t2

贝塞尔曲线的属性

  • 注意，这些属性在使用CAShapeLayer时，只会遵循CAShapeLayer的设置
  • 设置线条型号，- (void)setLineDash:(const CGFloat *)pattern count:(NSInteger)count phase:(CGFloat)phase
    • pattern：C类型的线型数据
    • count：pattern中的数据个数
    • phase：开始画线型的起始位置
  • 另外CAShapeLayer要和UIBezierPath结合在一起才有意义，单独使用意义不大

几个问题

  • 使用UIBezierPath绘制的曲线如何添加到当前的View当中？
    • 有两种方法：
      • 方法一：新建一个UIView，覆盖视图控制器自带的View，然后在新建的View当中进行添加便可
      • 方法二：绘制完成之后使用CAShapeLayer进行封装，就可以直接通过addSublayer的方法添加到当前的View当中
  • 可以给贝塞尔曲线添加动画效果吗？
    • 动画效果一种是给UIView添加，一种是给CALayer对象添加，并没有听过贝塞尔曲线添加
  • 类似于QuartzCore这样的工具什么时候去用比较合适？

贝塞尔曲线的弧度相关问题

  • 如何使用贝塞尔曲线画出弧线拼出一个圆？
    • 选择可以用三次贝塞尔曲线，也可以使用二次贝塞尔曲线
    • 使用三次贝塞尔曲线的方案：通过四条三次贝塞尔曲线来实现，每条贝塞尔曲线上面有两个控制点来控制曲线的弧度
      • 那么问题来了，如何通过两个控制点来控制曲线的弧度
      • 可以把圆放到一个正方形中去控制，通过然后让每条互相上的连个点关于对角线镜像对称，然后不断调整，直到出现完美的弧线为止
      • 调整的参数最好选择与边长或者对角线这样的数据关联起来，而不要通过猜测具体的数据来关联，否则兼容性就太差了，只会支持特定半径的圆，而无法兼容所有尺寸的圆，一般可以采用CGFloat offset = self.bounds.width/xxx的形式来测试调整参数，xxx指的是具体的数字
      • BTW，经过测试发现三次贝塞尔曲线画圆弧的最佳的xxx是3.6
    • 使用二次贝塞尔曲线的方案：基本方法同上，但是找不到最合适的xxx，所以使用二次贝塞尔曲线去画圆弧不是一个好的方案
  • 对贝塞尔曲线的误解
    • 我一直以为贝塞尔曲线画的弧线是会经过控制点的，然而实际上控制点一般是不会经过所画的圆弧的，一个特例是画直线的时候，其他时候都不会经过弧线
