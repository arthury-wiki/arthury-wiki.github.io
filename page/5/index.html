Nodejs01

浏览器的异步与事件触发

  • I/O阻塞与I/O不阻塞
    • I/O操作一般都具有阻塞性，也就是遇到I/O操作的时候，线程就会停在这个地方直到这个I/O操作执行结束之后才会继续执行接下去的任务，这就好像是工作的流程被I/O给堵塞掉了一样，进行不下去(因为浏览器是单线程的，所以会造成这样的阻塞)；当然如果不是I/O操作就不会造成阻塞的想象，这是为什么呢？
    • I/O的执行是在事件轮询之外进行的，当I/O操作结束之后，它会发出一个“事件”，这个时候会有一个回调函数来处理这个事件，这个事件就会跑到事件轮询当中去执行；这样就称I/O是异步的，并不会辅阻塞脚本执行
    • I/O异步可以让服务器始终处于能够响应事件的状态，与传统的阻塞性相比较，性能提高了不少，对于I/O负载比较重的Web程序，Node可以派上比较大的用场

QuartzCode02--Layer Basics

Layers Arrangement

  • Alignment
    • 为了能够一次性校对多种layer，你可以使用在toolbar上面的按钮。从左到右分别是左对齐、下对齐、垂直中心对齐、中心对齐、水平中心对齐、上对齐以及右对齐
    • 当多种layer被选中的时候，如果你点击任意的对齐按钮，这些layer将会与第一个被选中的layer对齐。按住command键当相机对齐按钮的时候来修改它的行为
    • layer也可以根据它们的anchor point来对齐。按住alt键当点击中心对齐按钮的时候来使用anchor point来对齐它们
  • Make Grid
    • 在QuartzCode中，你可以快速将layer布置到一个网格当中，通过使用Layers>Make Grid来进行实现…QuartzCode自动添加新的layer到那些layer当中落在了网格的区域 -
  • Arrange Layers on Path
    • 你也可以将layer布置到一个轨迹当中，通过使用Layer>Arrage Layers on Path；你可以在轨迹的弹出框中选择你已经创建好的任意形状

Selection

  • 添加layer
    • 想要添加layer，只需简单点击toolbar。在toolbar中选中了一个layer之后(可以点击shapes在弹出菜单当中进行选择)，在画布当中进行点击并且进行拖拽以此嵌入layer。可以按住shift键进行拖拽，这样就可以让拖拽的图形变得方方正正
  • 选择layers
    • 只需在画布当中点击便可；当然，你也可以在旁边的Layer List中进行选择
    • 想要选中当前选中layer的子layer，可以按下Enter键或者是在子layer上面进行双击
  • 选中多个layer
    • 这里有几个方法可以让你选中多个layer：
      • 1.最常用的拖拽鼠标覆盖自己要选中的layer，简单粗暴
      • 2.按住shift键去选择自己想要选中的layer
      • 3.使用上面两种方法在Layer List中进行操作
  • 移动layer
    • 一个被选中的later可以使用鼠标拖拽的方式进行移动，如果按住option键进行拖动，那么就会拖拽出一个副本
    • layers会被自动snapp到其他layer的控制点山，如果要关闭snap，可以在移动layer的时候按住command键
    • 一个被选中layer可以通过上下左右来进行移动，这样每次只会移动1px，如果按住了shift键进行就会移动10px(这一点倒是和sketch一样)
  • 调整layer的大小
    • 想要调整选中layer的大小，点击并且拖拽控制点就可以进行调整。如果在调整过程中想要维持layer的比例，那么可以按住shift键再进行拖拽。当被调整的layer拥有子layer的时候，它的子layer将不会跟着被调整，除非是group layer和effect layer

Sublayers and Grouping

  • 要让一个layer成为另一个layer的子layer，只需要在Layer List当中简单的拖拽一个layer到你想要作为父layer的layer中便可。任意的layer都可以作为其他layer的子layer
  • 如果要group多个layer，直接使用toolbar上面的group与ungroup便可。虽然组合与添加子layer看起来很相似，但是在QuartzCode当中有一些不同的地方
  • 但你组合多个layer的时候，layer将会被放置在一组的layer里面。一组的layer尺寸会随着子layer的尺寸变化而变化，一旦子layer的尺寸发生了变化，那么这一组的layer尺寸都会跟着变化

Rotation

  • 想要旋转layer，可以使用Layer Inspector中的rotation控制条来进行调整。默认的，这些layer是围绕着它的中心或者是{0.5,0.5}的anchor point。父layer的旋转会影响到它的子layer。
  • 为了改变layer在不同点的的旋转角度，可以在Layer Panel中改变anchor point。Anchor point在画布上就是一个蓝色的小标记，很好辨认

QuartzCode01

概述

  • QuartzCode是一个矢量绘图和动画app，可以帮助你绘制图形和制作动画，并且转换成Objective-C代码或者是Swift代码，QuartzCode对于开发iOS以及OSX开发者来说非常的重要

Layer List

  • Layerlist展示了所有的layer。这些layer是根据layer的继承等级展示出来的并且通过着色的缩略图的形式展示出来。你可以通过在Layerlist中改变layer的顺序来改变layer的继承等级
  • 所有的layer都向CALayer一样运作或者它们是QuartzCore的子类。取决于它们在QuartzCode中的layer类型，这些layer的行为可能不太一样
  • 你可以通过Layerlist选择layer或者高亮它们，你同样可以通过Layerlist隐藏或者锁定它们

Animation List

  • AnimationList是管理各种动画的地方，可以在这里添加或者是删除动画

Layer Inspector

  • Layer Inspector可以让你改变layer的属性设置，不同的layer类型有着不同的属性设置

Animation TimeLine

  • Animation TimeLine控制板展示了为layer准备的动画，它可以为所有的layer展示动画，为layer准备的动画只包含动画以及被选中的layer的动画，这些设置可以通过F1，F2以及F3来进行修改

Keyframe Inspector

  • Keyframe Inspector被用来改变动画以及它的keyframe设置

Sketch01

Layer介绍

  • handler
    • 图中白色的八个小方块就是handler
    • 可以利用这8个小方块来控制layer的形状
  • 移动layer
    • 可以选择layer不放开进行拖动
    • 可以选中之后使用上下左右按键来进行移动，使用这种方法可以每次只移动1个单位
    • 如果按住shift再使用上下左右按键来移动，每次就可以移动10个单位
  • duplicate
    • 选中layer然后右键菜单，点击duplicate就可以复制出一个一模一样的layer出来
    • 按住option键然后选中layer，这样就可以直接拖出当前layer的副本了，这是一个快捷方式
  • 隐藏layer
    • 如果不需要当前的layer，又不想删除它，就可以选择隐藏layer；在左边的layerlist中选中想要隐藏的layer，然后点击眼睛图标，就可以隐藏当前的layer了
  • 锁定layer
    • 当想要当前layer锁定在画布当中不被选中时，比如作为背景的时候，就可以考虑锁定layer，选中想要锁定的layer，然后右键选择lock layer便可，这样这个layer就不能够在画布当中被选中了，但是仍然可以在左边的layerlist中进行选择
  • group layer
    • 可以将选中的layer全部合成一组，这个操作很常见，这里就不详细介绍了
    • 想要操作一组layer里面的单个layer，可以在画布当中双击这个layer，这样就可以单独操作这个被选中的layer了
  • 显示layer之间的距离
    • 如果想知道选中layer与其他layer之间的距离，那么就可以按住option键然后将鼠标移到其他的layer上面，这样就可以看到距离提示了
  • 颜色的填充
    • 颜色的填充有很多的样式，具体的效果可以自己在fill选项卡当中进行选择，有纯颜色填充，有渐变，各种各样都有，具体效果的设计自己在实战当中去把握
    • 有一个很强大的功能要注意，就是点击上图中的试管可以对屏幕中任意位置进行取色，非常强大的功能，如果看到网上有什么好的颜色自己调不出来的，就可以覅用这个功能来取色，非常的方便
  • border属性
    • 这个属性是边框的样式，可以调整边框的颜色，宽度，颜色等等；点击齿轮按钮还可以设置更加细节的东西，比如说边框是否是虚线，是否为圆角等等都是可以设置的
  • shadow属性
    • 这个是layer的阴影属性
  • 拷贝CSS样式
    • 选中layer，右键可以拷贝当前layer的CSS样式，不过生成的代码貌似很糟糕，大多数是否都是自己去写比较合适
  • Blending
    • 通过这个属性可以查看颜色混合之后的效果，虽然不知道有什么卵用
    • 将一个layer的blending属性修改成非Normal之后就可以将这个layer覆盖到其他的layer上面查看混合的效果了

      画布

  • 移动画布
    • 按住空格键在点击画布的任意区域，就可以拖动画布了
  • show rulers
    • 可以通过tool bar中的views按钮选择这个功能，这样会在画布周围配上尺子来辅助布局
  • 放大与缩小画布
    • 除了可以使用tool bar上面的放大镜来进行缩放之外，还可以按住z键，然后点击任意你想要放大的区域；按住option+z，然后点击任意你想要缩小的区域
    • 还可以使用command +/-来放大与缩小；使用command 0来回到100%的比例

Pyspider02--html和css选择器

简单了解万维网

  • 万维网是一个由许多互相链接的超文本页面(也就是网页)组成的系统。
  • 网页使用网址(URL)进行定位，并链接彼此
  • 网页使用http协议传输
  • 网页使用html描述外观和语义

爬网页的流程

  • 1.找到包含我们需要的信息的URL列表
  • 2.通过http协议把页面下载回来
  • 3.从页面的html中解析出需要的信息
  • 4.找到更多的url，回到

选取一个Starter URL

  • 我们以爬去豆瓣电影作为入门教程
  • 既然我们要爬所有的电影，首先我们需要抓一个电影列表(也就是介绍电影的URL列表)，一个好的列表应该：
    • 包含足够多电影的URL
    • 通过翻页，可以遍历所有电影
    • 一个按照更新时间排序的列表，可以更快抓到最新更新的电影(或者是按照经典进行排序的电影，根据自己的需求来决定)

创建一个项目

  • 在pyspider的dashboard中点击”Create”按钮，命名项目为”tutorial_douban_movie”

  • 替换on_start函数的self.crawl的URL：

    @every(minutes=24 * 60)
def  on_start(self):
    self.crawl('http://movie.douban.com/tag/', callback=self.index_page)

  • self.crawl告诉pyspider抓取指定页面，然后使用callback函数对结果进行解析；每一个抓取都要配置一个指定的callback来进行解析

  • @every修饰器，表示on_start每天会执行一次，这样就能抓到最新的电影了
  • 点击run，就可以看到follows上面有一个红色的1，点击follows面板，再点击绿色的播放按钮便可

Tag列表页

  • 在tag列表页，我们需要提取所有的电影列表页的URL；目前sample handler一斤提取了非常多大的URL，所有，一种可行的提取列表页URL的方法就是使用正则表达式进行过滤：

    import re
...

    @config(age=10 * 24 * 60 * 60)
    def index_page(self, response):
        for each in response.doc('a[href^="http"]').items():
            if re.match("http://movie.douban.com/tag/\w+", each.attr.href, re.U):
                self.crawl(each.attr.href, callback=self.list_page)

  • 由于电影列表页和tag列表页不太一样，因此在这里新建以另一个callback作为self.list_page

  • @config(age=10 24 60 * 60)在这表示我们认为10天内页面有效，不会再次进行更新抓取(这里age表示多久之后需要重新抓取进行分析)
  • 由于pyspider是纯python环境，你可以使用python强大的内置库，或者你熟悉的第三方库对页面进行解析
